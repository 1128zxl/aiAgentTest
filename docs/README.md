# 错误处理文档

本目录包含项目的错误处理报告和文档。

## 文件说明

| 文件 | 说明 |
|------|------|
| `error-handbook.md` | 错误处理手册主文件（Markdown 格式） |
| `error-report-001-docker-desktop.html` | 错误 #001 的详细报告（HTML 格式） |

## 如何添加新错误

遇到新错误并解决后，可以：

1. **使用 AI 助手**：直接告诉 AI "帮我把这个错误记录到错误处理手册"
2. **手动添加**：参考 `error-handbook.md` 的格式添加新记录

## 查看报告

HTML 报告可以用浏览器直接打开：

```powershell
# 在浏览器中打开
Start-Process "docs\error-report-001-docker-desktop.html"
```
