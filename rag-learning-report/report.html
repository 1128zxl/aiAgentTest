🔄 完整调用流程

1
用户请求: POST /api/chat { question }
2
语义检索: 问题向量化 → 在向量库中查找最相关文档
3
构建提示词: 将检索结果与问题拼接成完整 prompt
4
调用智谱 API: 使用 glm-4-flash 生成回答
5
返回结果: { answer }

⚙️ 核心组件详解

📥

loader.ts

加载本地 Markdown/TXT 文件，封装为 LangChain Document 对象

✂️

splitter.ts

将大文档切分为 500 字符的小块，保留上下文重叠

📊

store.ts

内存向量存储，支持快速语义检索（可扩展到持久化存储）

🔗

chain.ts

核心 RAG 链：检索 → 提示词 → LLM → 输出

======== 问题: 年假有几天？ ========
回答: 员工入职满1年后，享有5天年假。入职满5年后，年假增加至10天。

======== 问题: 病假需要什么证明？ ========
回答: 病假需要提供医院开具的病假条。如果连续病假超过3天，还需提供三甲医院的证明。

======== 问题: 加班怎么算工资？ ========
回答: 工作日加班按1.5倍工资计算，周末加班按2倍，法定节假日按3倍。
